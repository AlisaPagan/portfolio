# Alisa Pagan Portfolio

Personal portfolio website built with **Next.js**, **React**, **TypeScript**, and **CSS Modules**.
It presents my front-end projects, UX/UI case studies, work experience, skills, and contact information.

**Live site:** https://alisapagan.dev

## Overview

This portfolio was designed and developed to support my transition into front-end development. It combines my background in UX/UI and graphic design with modern front-end implementation, responsive layouts, accessibility improvements, and performance optimization.
<img width="1498" height="1047" alt="image" src="https://github.com/user-attachments/assets/ecd39a78-72c9-4c17-8414-01a4be5ba601" />


## Features

* Responsive design for desktop, tablet, and mobile
* Dynamic project detail pages
* Front-end project and UX case study sections
* Dark/light theme toggle
* Animated section reveals
* Image sliders for project screenshots and mockups
* Contact form with toast notifications
* Custom 404 page
* SEO metadata and Open Graph setup
* PageSpeed-focused image and layout optimization

## Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **UI:** React, CSS Modules
* **Libraries:** Swiper, React Hot Toast, React Icons
* **Analytics:** Vercel Analytics
* **Deployment:** Vercel

## Main Sections

* **Home** — hero section, quick intro, and navigation
* **About** — background and career focus
* **Skills** — front-end, back-end/API, design, and tools
* **Projects** — featured development and UX/UI work
* **Experience** — design, web, and project background
* **Ongoing & Learning** — current projects and learning path
* **Contact** — contact form and direct links

## Featured Work

* **Relax Map** — team-built full-stack platform for discovering and reviewing relaxation locations

* **Mebleria** — responsive furniture store web app built as a JavaScript team project

## What I Focused On

* Building a polished front-end experience with reusable components
* Keeping the design responsive and consistent across screen sizes
* Improving accessibility with semantic structure, ARIA labels, and keyboard-friendly controls
* Optimizing images, loading behavior, and layout stability
* Presenting both development work and UX/product thinking in one place

## Run Locally

```bash
git clone https://github.com/AlisaPagan/portfolio.git
cd portfolio
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
